# project-friis

### AI/ML-Based Wireless Signal Strength Modeling

## Overview

Project-Friis explores **wireless signal propagation analysis** using the **Friis Transmission Equation** and extends it toward **AI/ML-driven signal strength prediction and optimization**.

The project simulates RF signal attenuation with distance and generates datasets that can later be used to train **machine learning models for wireless propagation prediction and intelligent network optimization**.

---

## Features

* Friis transmission equation based signal strength simulation
* Received power vs distance analysis
* Parameterized RF modeling (power, frequency, antenna gain)
* Dataset generation for machine learning experiments

---

## Future AI/ML Extensions

* ML models for **signal strength prediction**
* **Obstacle-aware propagation modeling**
* **Physics-informed neural networks for RF systems**
* **Reinforcement learning for antenna placement and network optimization**
* AI-driven **wireless coverage prediction**
